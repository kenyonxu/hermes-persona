## 使用方法

1. 选择上述示例之一，复制 JSON 内容
2. 保存为 `persona-config.json`，放到 Hermes profile 目录（如 `~/.hermes/profiles/default/`）
3. 重启 Hermes gateway 使配置生效
4. 发送任意消息，Agent 将在系统提示中包含对应人格上下文

如需微调，直接修改 JSON 文件中的字段即可——无需改动任何 Python 代码。
